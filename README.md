# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/SwarajThakre/Personal_portfolio.git
```

Navigate to the project

```bash
cd Personal_portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
